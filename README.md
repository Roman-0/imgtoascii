# imgtoascii
Converts images into ascii art. 
```
USAGE:
  (in directory of imgtoascii)
  1. cargo run --release -- --path <FILE>

FLAGS:
    --path, -p (required)
            path to the image
    --resolution, -r (default = 32)
            x resolution of the generated text
